HTML

HTML Tables

Table Tags

Tag Name Description
<table> Table The <table> HTML element represents tabular data in a two-dimensional table comprised of rows and columns of cells containing data.
<thead> Table Head The <thead> HTML element groups together a set of table rows (<tr> elements) that make up the table's header. Generally, containing columns headers (<th> elements) that provide information about the table's columns.
<tbody> Table Body The <tbody> HTML element contains a group of table rows (<tr> elements), representing the main data section of a table.
<tr> Table Row The <tr> HTML element specifies a row within a table. And this row can contain a combination of <td> (data cell) and <th> (header cell) elements to defines its individual cells.
<td> Table Data The <td> HTML element specifies a data-containing cell within a table and can be used as a child of the <tr> element.
<tfoot> Table Foot The <tfoot> HTML element contains a set of <tr> elements, signifying that they make up the footer of a table with details about the table's columns. This section typically provides a summary, such as the total of the numbers in a columns.

Table Attributes

Attribute Name Description
colspan Column Span Specifies the number of columns a <td> element should span.
rowspan Row Span Specifies the number of rows a <td> element should span.
